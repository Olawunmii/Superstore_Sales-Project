# Superstore_Sales-Project
I will be working on Super Store Sales Data on Kaggle-https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting
Global Superstore Sales Forecasting

## Project Overview

This project analyzes four years of retail sales data from a global superstore to uncover business insights and forecast sales for the next seven days using time series forecasting techniques.

The objective is to demonstrate the complete data analytics workflow, including data cleaning, exploratory data analysis (EDA), visualization, feature engineering, and predictive modeling.

---

## Business Problem

Retail businesses need accurate sales forecasts to improve inventory planning, staffing, and business decision-making. This project develops a forecasting model capable of predicting future sales based on historical trends.

---

## Dataset

Source: Global Superstore Dataset

The dataset contains four years of transactional retail data, including:

- Order Date
- Ship Date
- Sales
- Profit
- Quantity
- Customer Information
- Product Category
- Region
- Segment
- Shipping Mode
